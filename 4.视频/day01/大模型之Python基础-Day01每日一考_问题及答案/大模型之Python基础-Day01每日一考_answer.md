### 题目 1：环境相关

在安装好 Python 环境后，打开命令行窗口，输入什么命令可以查看当前安装的 Python 版本？

**答案**：python --version （在 Windows 系统中，如果安装时未勾选将 Python 添加到环境变量，可能需要进入 Python 安装目录下执行该命令；在 Linux 或 macOS 系统中，一般直接在终端输入即可）

**分析**：这道题考察对 Python 环境搭建后基本命令的使用，查看 Python 版本是确认环境安装是否成功以及了解当前 Python 环境信息的重要方式。

### 题目 2：基础语法

以下 Python 代码有什么错误？

```python
print('Hello World)
```

**答案**：字符串缺少右引号，正确代码应为 print('Hello World')

**分析**：考察学生对 Python 中字符串表示方式的掌握，字符串需要用引号（单引号或双引号）完整包裹起来，否则会导致语法错误。

### 题目 3：程序实现

编写一个 Python 程序，输出自己的名字。

**答案**：

```python
print('你的名字')
```

将你的名字替换为学生自己的名字即可。例如：

```python
print('张三')
```

**分析**：这道题基于第一天输出Hello World程序进行简单拓展，让学生熟悉使用print函数输出不同内容，加深对简单 Python 程序结构的理解 。
