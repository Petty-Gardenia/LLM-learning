1.函数
    直观的理解：带名字的代码块
    将完成某一个功能的代码抽取出来，取一个名字，就是函数。如果以后再要完成这个功能的时候，不需要在编写整体代码了，直接通过
    名字去调用函数即可。

2.函数的定义
    语法：
        def 函数名(参数列表):
            函数体
            [return]

    说明：
        def是定义函数的关键词
        函数名，程序员自己定义，需要遵循标识符的命名规范，建议望文生义
        参数列表
            函数名后，跟着一对小括号，里面放的是参数列表
            参数：函数执行的时候需要依赖的值
            参数可以有多个，多个参数之间用逗号分隔
            如果没有参数，小括号也不能省略
        用:结束函数的声明
        函数具体完成的功能在函数体中实现，需要进行缩进
        通过return关键字将函数执行的结果返回给调用者，可以不加，默认返回值是None
    注意：
        函数必须先定义再调用
        函数只有在调用的时候才会被执行。在定义的时候只是告诉解释器我定义了一个这样的函数，可以完成某些功能，但是这个时候函数还没有执行

3.定义函数的好处
    使程序变得更简短而清晰
    可以提高程序开发的效率
    提高了代码的重用性
    便于程序分工协作开发
    便于代码的集中管理
    有利于程序维护

4.形式参数和实际参数
    在函数的提供者定义函数的时候，声明的参数称之为形式参数(形参)
        def print_star(rows,cols):
    在函数的调用者调用函数的时候，传递的参数称之为实际参数(实参)
        print_star(2,3)

    注意：在函数声明的时候，并没有给形式参数分配内容空间，只是做了一个占位。
         在调用的时候，会将实参的地址引用赋值给形参，相当于做了如下操作
            rows,cols = 2,3

5.函数参数的传递
    传递不可变类型参数
        如果在调用函数的时候，参数传递的是不可变类型数据，在函数体中，如果对参数进行修改，不会在原对象上进行修改，会创建新的对象。
    传递可变类型参数
        如果在调用函数的时候，参数传递的是可变类型数据，在函数体中，如果对参数进行修改，会直接在原对象上进行修改，不会创建新的对象。

    注意：
        对于参数传递的是可变数据类型
            var1 *= 2使用原地址。
            var1 = var1 * 2开辟了新的空间。
        对于参数传递的是不可变数据类型
            var1 *= 2  、var1 = var1 * 2开辟了新的空间。

6.参数的传递形式
    必须参数
        参数的个数和顺序要一致，通过位置进行赋值
    关键字参数
        传递实参的时候要带名称，并需要和形参名字一致，位置可以不一致   通过参数名称进行赋值
    默认值参数
        在定义参数的时候，可以给参数指定默认值
        在调用函数的时候，带默认值的参数可以不传递(形参的数量和实参的数量可以不一致)
        如果在调用函数的时候，没有给默认值参数传参，就使用默认值，如果给默认值参数传参了，就会用传递的参数覆盖默认值
        声明在必须参数后
    不定长参数
        方式1：def 函数名([普通参数1],*不定长参数名,[普通参数2])
            在调用函数的时候，普通参数1（不定长参数前），按照位置进行匹配
            在调用函数的时候，普通参数2（不定长参数后），按照关键字进行匹配
            在处理不定长参数的时候，是使用元组进行处理。

        方式2：def 函数名([普通参数1],**不定长参数名)
            在调用函数的时候，普通参数1（不定长参数前），按照位置进行匹配
            不定长参数后面不允许出现其它的参数
            在处理不定长参数的时候，是使用字典进行处理。

    解包传参
        调用方法传递参数的时候，可以 (*(元组、列表、set))
        调用方法传递参数的时候，可以 (**{字典})

7.return
    结束当前正在执行的方法，返回方法的调用位置
    返回方法的执行结果给调用者

    如果函数中没有return或者return后什么也没有，默认返回的是None
    return 后可以返回多个结果,会将多个结果封装为元组
