### 题目 1：分支语句

编写一个 Python 程序，输入一个整数，判断它是正数、负数还是零，并输出相应的结果。

```python
num = int(input("请输入一个整数："))
if num > 0:
    print("这个数是正数")
elif num < 0:    
    print("这个数是负数")
else:
    print("这个数是零")
```

**答案**：上述代码通过if-elif-else分支语句，根据用户输入的整数与 0 的比较结果，输出对应的判断信息。

**分析**：本题考查学生对分支语句的运用，if、elif和else的条件判断结构，让学生学会根据不同条件执行不同代码块。

### 题目 2：循环语句

使用for循环打印出 1 到 10 的整数。

```python
for i in range(1, 11):
    print(i)
```

**答案**：利用range(1, 11)生成从 1 到 10 的整数序列，for循环遍历该序列并依次打印每个整数。

**分析**：检验学生对for循环的理解，包括range函数的使用，通过循环依次执行打印操作，强化循环结构的应用。

### 题目 3：分支与循环结合

编写一个程序，从 1 循环到 10，当数字是偶数时打印 “偶数”，奇数时打印 “奇数”。

```python
for num in range(1, 11):
    if num % 2 == 0:
        print("偶数")
    else:
        print("奇数")
```

**答案**：for循环遍历 1 到 10 的数字，在每次循环中，通过if-else分支语句判断数字是奇数还是偶数，并打印相应结果。

**分析**：该题综合考查分支语句和循环语句的结合使用，让学生明白如何在循环过程中根据条件执行不同操作，提升代码综合运用能力。

### 题目 4：循环控制

编写一个while循环，计算 1 到 100 的整数之和。

```python
sum_num = 0
i = 1
while i <= 100:
    sum_num += i
    i += 1
print(sum_num)
```

**答案**：初始化和变量sum_num为 0，计数变量i为 1，通过while循环，只要i小于等于 100，就将i累加到sum_num中，并使i自增 1，循环结束后打印出 1 到 100 的整数和。

**分析**：本题考察while循环的使用，以及在循环中通过条件控制和变量自增实现特定计算任务，加深对循环控制机制的理解。
